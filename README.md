# Be Alert
- 재난경보 PWA앱
- 경보 소리가 너무 시끄러워서 꺼놨는데 그래도 알람은 필요할듯하니 상냥한 알림을 작성해보자

# 스택
- Nextjs
- Shadcn/ui
- Fast API
- FCM
- MariaDB
